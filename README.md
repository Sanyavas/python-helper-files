# python-helper-files

Repository with reusable Python helper files for new projects.

## Files

- `py_logger.py` — logger with console output, file logging, colors, and log rotation
- `file_reader.py` — helper for reading content from different file formats such as PDF, TXT, MD, JSON, and DOCX
- `file_writer.py` — helper for saving data to different file formats such as JSON, TXT, and PDF

## Purpose

Store useful helper files in one place so they are easy to find and reuse in future projects.
